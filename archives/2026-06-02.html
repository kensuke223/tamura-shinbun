「ザ・スーパーマリオギャラクシー・ムービー」洋画No.1達成　全世界興収1,326億円の大ヒット
イルミネーションとニンテンドーが贈るマリオ最新アニメ映画が快記録を更新中。日本国内では今年公開の洋画作品No.1を獲得し、興収60億円を突破。全世界での累計興行収入は約1,326億円（9億ドル超）に達し、2026年公開アニメ映画最大のヒット作となっている。先週末も国内ランキング上位をキープし、ロングランヒットの勢いが止まらない。

NiziU 東京ドーム千秋楽をHuluが独占ライブ配信　約3年半ぶりドームツアー

Microsoft Build 2026本日開幕　WindowsをAIエージェントPFに宣言　Project Polaris発表
本日6月2日、サンフランシスコでMicrosoft Build 2026が開幕。Satya Nadella CEOはWindowsをAIエージェントのファーストクラス実行環境と宣言し、Windows Agent Framework（WAF）をMITライセンスでオープンソース化した。また自社開発AIコーディングモデル「Project Polaris」を発表。2026年8月よりGitHub CopilotのデフォルトエンジンとしてGPT-4 Turboを置き換える計画で、OpenAI依存からの脱却を鮮明にした。
